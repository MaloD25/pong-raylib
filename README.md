\# Pong Raylib



A simple Pong game made in C++ with raylib.



This project is my first real game made with raylib.  

I already had some basic knowledge of C++, but this project helped me put that knowledge into practice in a real game project.



\## Project goal



The goal of this project was to learn the basics of game development with raylib and improve my C++ practice through a complete mini-game.



\## Features



\- Main menu

\- Solo mode against a simple AI

\- 2-player mode

\- Score system

\- End of match screen

\- Animated menu background

\- Ball bounce angle changes depending on where it hits the paddle



\## What I learned



Through this project, I practiced and improved:



\- the basics of raylib

\- game loops

\- classes and project organization

\- enums and game states

\- separating code into multiple files

\- using C++ in a more concrete way through a game project



I also tried to organize the project as well as possible by separating the code into different classes, even if I am still learning what good project organization really looks like.



\## Main difficulties



The most difficult part of the project was managing the different game states.



Having multiple states in the game was not easy to organize, and it was also a source of bugs.  

The gameplay itself was not the hardest part for me, but building a clean structure for the menu, the gameplay, the pause between points, and the end-of-match screen was more challenging.



\## How to run



Open the project in Visual Studio and press the green run button.



\## About the game



The game is simple and small.



It starts with a menu where the player can choose:

\- 1 player against an AI

\- 2 players



Then the match starts and the players can play until one side wins.
