\# Pong Raylib



My first Pong game made in C++ with raylib.



\## Features

\- Solo mode against a simple AI

\- 2-player mode

\- Main menu

\- Animated menu background

\- Score system

\- Game over screen



\## Technologies

\- C++

\- raylib

\- Visual Studio



\## What I learned

\- game loop

\- collision handling

\- game states

\- menu system

\- simple AI

\- project organization in multiple files
